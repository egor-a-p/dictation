ДИКТАНТ

О сервисе:

Данный проект выполняется в целях изучения технологий и не несет сильной смысловой нагрузки=)

Что сделано:

  • каркас приложения с помощью Spring Boot
  • cинтез речи с помощью Yandex SpeechKit Cloud;
  • сохранение/выгрузка контента в MySQL DB;
  • фильтр по авторам;
  • веб-интерфейс с использованием Thymeleaf и bootstrap.

Что делается:

  • пользователи и роли;
  • добавить сложность диктантов (4-й класс, 5-й класс и т.д.) и фильтр по сложности;
  • улучшить синтез речи для диктантов: помедленнее + повторы;
  • поработать с веб интерфейсом, добавить рюшечки из jQuery.
